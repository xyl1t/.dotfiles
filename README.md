## Installation

If you are me or really like my dotfiles and want to install them,
you will need GNU stow, once installed, follow these steps:

```sh
# clone repo
git clone https://github.com/xyl1t/.dotfiles.git
cd .dotfiles
# install dotfiles
stow -vt ~ */
```
